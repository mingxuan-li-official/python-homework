========================================
图书管理系统 - 快速开始指南
========================================

一、环境准备

1. 安装MySQL数据库
   - 请确保已安装MySQL 5.7+ 或 MySQL 8.0+
   - 启动MySQL服务

2. 配置数据库连接
   - 编辑 config.py 文件
   - 修改MySQL连接信息：
     * host: MySQL服务器地址（默认localhost）
     * port: MySQL端口（默认3306）
     * user: MySQL用户名（默认root）
     * password: MySQL密码（请修改为实际密码）
     * database: 数据库名称（默认library）

3. 安装Python依赖
   - 在命令行运行: pip install -r requirements.txt
   - 或: pip install mysql-connector-python

二、启动步骤

1. 启动服务端
   - 双击运行 start_server.bat
   或
   - 在命令行运行: python server.py
   
   服务端启动后会显示: "图书管理系统服务端已启动，监听 0.0.0.0:8888"
   首次运行时会自动创建数据库和表结构

2. 启动客户端
   - 双击运行 start_client.bat
   或
   - 在命令行运行: python main.py
   
   客户端启动后会显示登录界面

二、登录说明

默认管理员账户：
- 用户名: admin
- 密码: admin123

首次使用：
1. 点击"连接到服务器"按钮
2. 输入用户名和密码
3. 点击"登录"按钮

注册新用户：
1. 点击"注册"按钮
2. 填写用户信息（用户名、密码、姓名、角色等）
3. 点击"注册"完成

三、功能说明

管理员功能：
- 图书管理：添加、编辑、删除、搜索图书
- 借阅管理：查看所有借阅记录，按状态筛选
- 统计信息：查看系统统计数据

普通用户功能（会员/普通用户）：
- 图书浏览：搜索和浏览所有图书
- 图书借阅：借阅可用的图书
- 我的借阅：查看自己的借阅记录，归还图书
- 个人信息：修改个人信息和密码

四、注意事项

1. 必须先启动MySQL服务，再启动服务端
2. 必须先启动服务端，再启动客户端
3. 服务端和客户端可以在同一台机器或不同机器上运行
4. 如果客户端和服务端在不同机器，需要修改 network_client.py 中的 host 参数
5. 数据库和表结构会在首次运行时自动创建
6. 请确保 config.py 中的MySQL密码配置正确
7. 默认借阅期限为30天

五、常见问题

Q: 数据库连接失败？
A: 请检查：
   - MySQL服务是否已启动
   - config.py 中的连接信息是否正确
   - MySQL用户是否有创建数据库的权限

Q: 连接服务器失败？
A: 请确认服务端已启动，并且端口8888未被占用

Q: 忘记管理员密码？
A: 可以在MySQL中删除users表中的admin用户，重新启动服务端会自动创建默认管理员账户

Q: 如何修改服务器端口？
A: 修改 server.py 中的 port 参数，同时修改 network_client.py 中的 port 参数

Q: 如何重置数据库？
A: 在MySQL中删除library数据库，重新启动服务端会自动创建
